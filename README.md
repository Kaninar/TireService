
# Шиномотажки

Вебсайт, серверная часть

## Требования

- [Composer](https://getcomposer.org/)
- [npm](https://docs.npmjs.com/downloading-and-installing-node-js-and-npm)
- [PostgreSQL](https://www.postgresql.org/)

## Развёртывание

Перейти в директорию проекта
```
  cd path/to/project
```

Запуск на **Linux**:
```bash
  chmod +x deploy.sh
  deploy.sh
```
или

Запуск на **Windows**
```cmd
  sh deploy.sh
```
